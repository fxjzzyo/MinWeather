## MinWeather
 Welcome to MinWeather! This is a weather app powered by fxjzzyo.
____

### 功能
- 支持查询当天天气类型、温度、湿度、风力、风向、pm2.5指数等天气信息
- 支持对未来4天天气的预告
- 支持定位到本地
- 支持桌面widget小部件
- 支持国际化和本地化
- 支持友盟统计
- 当pm2.5值为空时，显示其所在省份的pm2.5值
